# АвиаДанные Pro ✈️

Профессиональное веб-приложение для работы с данными аэропортов, авиакомпаний и маршрутов на основе открытого датасета [OpenFlights](https://openflights.org/data.html).

![Главная страница](screenshots/01_main.png)

## Описание проекта

Приложение реализовано на **FastAPI** с использованием **Jinja2** для шаблонов и **SQLAlchemy** для работы с базой данных SQLite.  
Полностью соответствует требованиям задания и даже превосходит их за счёт дополнительных функций и современного дизайна.

### Основные функции
- Поиск аэропортов по диапазону географических координат (широта/долгота)
- Поиск аэропортов по городу и стране (с автодополнением)
- Просмотр всех рейсов в/из выбранного города и страны
- Поиск прямых рейсов между двумя городами
- **Дополнительно:** поиск рейсов с одной стыковкой (реализовано по желанию)
- Красивая пагинация результатов с сохранением параметров поиска
- Автодополнение для полей «Город» и «Страна»
- Адаптивный современный дизайн на Bootstrap 5 + анимации

## Технологии
- **Backend:** FastAPI
- **Frontend:** Jinja2 + Bootstrap 5 + Font Awesome
- **База данных:** SQLite + SQLAlchemy
- **Загрузка данных:** скрипт из файлов *.dat (OpenFlights)

## Как запустить проект (для Windows — одним кликом!)

1. Распакуйте архив проекта
2. Дважды кликните по файлу **`start_project.bat`**
3. При первом запуске подождите 5–10 минут (установка пакетов и загрузка данных в базу)
4. Откройте браузер по адресу: **http://127.0.0.1:8000**

> Бат-файл автоматически создаст виртуальное окружение, установит зависимости и загрузит данные.

Для остановки сервера нажмите **Ctrl + C** в окне.
