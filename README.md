# АвиаДанные Pro ✈️

Профессиональное веб-приложение для работы с данными аэропортов, авиакомпаний и рейсов на основе датасета OpenFlights.

![Главная страница]

## Функции
- Поиск аэропортов по географическим координатам
- Поиск аэропортов по городу и стране (с автодополнением)
- Просмотр всех рейсов в/из города
- Поиск прямых рейсов между городами
- **Дополнительно:** поиск рейсов с одной стыковкой
- Красивая пагинация и адаптивный дизайн
- Автодополнение для городов и стран

## Технологии
- **Backend:** FastAPI
- **Frontend:** Jinja2 + Bootstrap 5 + Font Awesome
- **База данных:** SQLite + SQLAlchemy
- **Загрузка данных:** из файлов OpenFlights (.dat)

## Как запустить (одним кликом!)

1. Распакуйте архив
2. Дважды кликните по `start_project.bat`(если бд не будет создана, до пропишите в терминале python -m load_data и запустите .bat ещё раз)
3. Подождите (первый запуск ~10 минут — установка и загрузка данных)
4. Откройте браузер: http://127.0.0.1:8000

> Батник автоматически создаст виртуальное окружение, установит пакеты и загрузит данные.
